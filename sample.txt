в проекте используем схему для postgre - performance_review_schema
и там есть пользователь review_user c паролем review_password у него есть все права
В корне проекта есть файл temporary_reset.py он удаляет таблицы из бд и заново создает(на всякий случай)


запуск бека
1 Gишешь в терминал python3 -m venv venv - Это создаст виртуальное окружение проекта для установки зависимостей(Для Linux).
                    python -m venv venv - Это создаст виртуальное окружение проекта для установки зависимостей(Для Windows).
(что бы в систему не ставились)(или и для того и для того python3 писать не помню уже)


2 В Терминале пишешь source venv/bin/activate - Это активирует виртуальное окружение(для Linux).
                     source venv/script/activate Это активирует виртуальное окружение(для Windows).
( что бы можно было устанавливать в него зависимости) если нужно выключить, то пишеш deactivate.

3 В терминале пишешь pip install -r requirements.py - Это установит все зависимости в виртуальное окружение.
(что бы ручками каждую зависимость не устанавливать)

Сервер запускается командой - uvicorn app.main:app --reload.
Выключается CTRL + C.
